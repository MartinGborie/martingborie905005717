This is the official (demo) website for the Leone Stars Football Academy, built using HTML, JavaScript, and Tailwind CSS.

## Project Goal
The goal of this project is to provide a professional, responsive, and engaging platform to:
1.  Showcase the academy's mission and vision.
2.  Highlight promising young players.
3.  Display upcoming match schedules and past results.
4.  Provide contact and community outreach information.

## Deployment Status
The live version of this website is automatically deployed via GitHub Pages from the `main` branch.

## Getting Started

To run this website locally, you only need to open the `index.html` file in your web browser.

### Technologies Used
* **HTML5**
* **Tailwind CSS** (via CDN for utility-first styling)
* **JavaScript** (for mobile menu and view switching)
* **Lucide Icons** (for iconography)
